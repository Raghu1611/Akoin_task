# Akoin_task

## About the Project
This project is a modern, responsive single-page website for the **Institute of Digital Risk (IDR)**. IDR is an industry-led training and deployment institute aimed at preparing professionals for digital, cyber, and technology risk roles in regulated and high-consequence environments. 

The website highlights IDR's mission to bridge the gap between theoretical knowledge and practical execution through academic collaboration and industry-led training.

### Key Features:
- **Responsive Design**: Built with HTML, CSS (Flexbox/Grid), and vanilla JavaScript.
- **Service Pillars**: Details on IDR's Academy, Innovation & Incubation, and Advisory Services.
- **Pipeline Model**: A visual representation of the Train → Hire → Innovate → Deploy model.
- **Community Focus**: Information on serving individuals (students, professionals) and organizations.
- **Contact Form**: A functional registration form system for interested parties.
